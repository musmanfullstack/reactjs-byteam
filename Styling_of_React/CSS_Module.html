---CSS Modules "Css Modules provides the safest as well as easiest way to define the style. It uses normal css stylesheet with normal syntax. While importing the styles, CSS modules convert all the styles into locally scoped styles so that the name conflicts will not happen. Let us change our component to use CSS modules. Next, create a new stylesheet, Expense.module.css file under src/components folder and write regular css styles." ---Styling "div.itemStyle { color: 'brown'; font-size: 14px; }" "Next, open Expense.js file in the src/component folder and import the styles. import styles from './ExpenseEntryItem.module.css' Next, use the styles as JavaScript expression in the component.
